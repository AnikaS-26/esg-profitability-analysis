# ESG Scores & Corporate Profitability Analysis

## Overview
Analyzed the relationship between ESG (Environmental, Social, Governance) scores and corporate profitability to understand whether higher ESG performance correlates with better financial outcomes. Used R for data cleaning, visualization, and statistical modeling.

## Tools & Technologies
- R (tidyverse, ggplot2)
- Statistical methods: regression analysis, data preprocessing (winsorization, scaling)

## Data
- Source: ESG and financial metrics dataset from Kaggle
- Key fields: ESG composite scores, revenue, profit margins, sector classification

## Steps
1. **Data cleaning & preprocessing**  
   - Removed duplicates and outliers  
   - Winsorized extreme values and scaled numeric variables  
   - Constructed analysis-ready dataset with ESG scores and profitability metrics  

2. **Exploratory data analysis**  
   - Visualized ESG score distributions and profitability across sectors  
   - Examined correlations between ESG dimensions and financial performance  

3. **Statistical modeling**  
   - Estimated regression models quantifying the relationship between ESG scores and profit margins  
   - Assessed effect sizes, statistical significance, and sector-specific patterns
